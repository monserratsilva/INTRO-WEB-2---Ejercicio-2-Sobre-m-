Descripción de términos: función, let, const y etiquetas HTML
Función:
Es un bloque de código que realiza alguna operación.
Let:
Es una variable que contiene un valor, es posible reasignar el valor a la variable declarada con este.
Const:
Es una variable que contiene un valor, no es posible reasignar el valor a la variable declarada con este.
3 etiquetas HTML:
"abbr" para abreviaturas; "br" para salto de línea sencillo; "map" para mapa de imagen.
